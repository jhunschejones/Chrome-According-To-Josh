# Collapse All Tab Groups

### Overview
This extension allows you to instantly collapse all tab groups in your current Chrome window, helping you reduce visual clutter and stay organized. Trigger the collapse with a keyboard shortcut or by clicking the extension icon.

### How to Use
1. Install the extension via `Load unpacked` in `chrome://extensions/`.
2. (Optional) Set your preferred keyboard shortcut under `chrome://extensions/shortcuts`.
3. Click the extension icon or use your shortcut to collapse all tab groups in the current window.

---

### Why?
Chrome tab groups are great, but collapsing each group manually is slow. This tool gives you one-click (or one-key) cleanup.
